My Projects

1. Harvesting Electricity from Road

Developed a system to generate electricity from road vibrations using piezoelectric sensors. This project aimed at renewable energy generation through innovative means.

This approach can potentially harness energy from the existing road infrastructure, contributing to sustainable energy solutions.

2. Automated Garbage Disposal System

Created an automated system using Arduino to detect garbage levels and activate disposal mechanisms. This project enhances urban hygiene by automating waste management.

By utilizing sensors, the system optimizes collection routes and schedules, reducing operational costs.

3. IMDB Score Prediction

Implemented a machine learning model using Python and Kaggle datasets to predict movie ratings. The project utilized algorithms such as linear regression and decision trees for accurate predictions.

This project aims to assist movie producers in making data-driven decisions based on audience preferences.

4. Smart Door Lock System

Designed an electronic door lock system with a keypad, allowing users to unlock the door with a password. The system features an LCD for user interaction, enhancing security.

The system integrates real-time monitoring, allowing users to control access remotely via a mobile application.

5. Weather App

Developed a weather application using HTML, CSS, and JavaScript that fetches real-time weather data from an API, providing users with accurate and timely weather updates.

This app allows users to plan their activities based on current weather conditions and forecasts.
